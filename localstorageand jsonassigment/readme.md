# Enhanced Todo List (LocalStorage + Search)

This project is an enhanced Todo List application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks  
- Mark tasks as completed (visually shown using line-through)
- Delete tasks  
- Search tasks in real time  
- Tasks stored in `localStorage` as a JSON array  
- Each task contains:
  - Unique ID  
  - Task text  
  - Completed status  
- Tasks persist even after refreshing the page

##  How to Use

1. Type a task in the input field and click **Add Task**.
2. Your task will appear in the list below.
3. Click **Done** to mark a task as completed.  
4. Click **Undo** to revert it.
5. Click **Remove** to delete the task.
6. Use the **Search bar** to filter tasks instantly.
7. All tasks automatically save in your browser via `localStorage`.
